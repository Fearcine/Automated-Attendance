How the Application Works

Register student faces using the camera

Store facial data securely in the database

Start attendance during lectures

Detect and recognize faces in real time

Mark attendance automatically with timestamps

View attendance records in Excel format


Installation and Setup

Clone the repository
```
git clone https://github.com/Fearcine/Automated-Attendance
```
Navigate to the project directory
```
cd Automated-Attendance

```
Create and activate a virtual environment

Install dependencies
```
pip install -r requirements.txt
```
Run the application
```
python app.py
```
